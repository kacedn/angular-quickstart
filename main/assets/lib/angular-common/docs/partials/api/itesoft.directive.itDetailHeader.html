itDetailHeader
directive in module itesoft

Description

A container element for detail header, MUST be include in <it-detail> . for more information see <it-master-detail>.

<it-master-detail>
  <!-- Master Content content -->

  <it-master>
      <it-master-header>
      </it-master-header>
  </it-master>

  <!-- menu -->
  <it-detail>
       <it-detail-header>
          <button class="btn btn-primary" title="Add" ng-disabled="currentItemWrapper.hasChanged" ng-click="myAction()"><span class="fa fa-plus fa-lg"></span></button>
      </it-detail-header>

      <it-detail-content>
      </it-detail-content>
  </it-detail>

</it-master-detail>

Usage

as element:
<it-detail-header>
</it-detail-header>
as attribute
<ANY it-detail-header>
   ...
</ANY>
